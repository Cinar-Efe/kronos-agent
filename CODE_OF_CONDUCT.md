# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Positive behavior includes using welcoming language, being respectful, and accepting constructive criticism.

Unacceptable behavior includes harassment, trolling, and publishing others private information.

## Enforcement

Instances of unacceptable behavior may be reported by opening a GitHub issue.
