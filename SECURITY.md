\# Security Policy



\## Supported Versions



| Version | Supported |

|---------|-----------|

| 0.1.x   | ✅ Yes    |



\## Reporting a Vulnerability



If you discover a security vulnerability, please \*\*do not\*\* open a public issue.



Instead, email directly or open a private security advisory:



1\. Go to the \[Security tab](https://github.com/Cinar-Efe/kronos-agent/security)

2\. Click \*\*"Report a vulnerability"\*\*

3\. Fill in the details



We will respond within \*\*48 hours\*\* and release a fix as soon as possible.



\## What to Include



\- Description of the vulnerability

\- Steps to reproduce

\- Potential impact

\- Suggested fix (optional)



\## Scope



\- Source code in this repository

\- npm package `kronos-agent`



Thank you for helping keep Kronos secure! ⌛
